When you click on the [Filter button], the text inside the Input area will be edited and the result will be displayed in the Output area.

[Replace] can be used to find all matched words with the target and replace them with 'As', including spaces.

[Insert] is useful for inserting characters periodically, such as when tracking codon units in DNA sequences.

[Extract] can be used to extract data based on target length, while the [Extract(Odd)] and [Extract(Even)] functions extract words positioned at (2N-1)th and (2N)th, respectively. These extract functions are useful for processing exCGen data and merging them into a longer sequence made only of + or - strands.

[Compare] tracks which characters have been changed.

[Send to Input] sends the result back to the Input area for further editing.
